# Banking Analytics Capstone Project

## Project Overview

This project presents an end-to-end banking analytics solution built using Power BI and MySQL. The analysis focuses on customer behavior, deposit trends, transaction activity, loan portfolio risk, card usage, and complaint resolution performance. Interactive dashboards and SQL analysis were used to generate actionable business insights.

## Tools & Technologies

- Power BI
- MySQL
- SQL
- Excel

## Dataset

The dataset includes the following tables:

- Customer
- Account
- Transaction
- Loan
- Card
- Complaint
- Date

- ## Power BI Dashboards

The project contains the following dashboards:

- Executive Overview
- Customers Dashboard
- Accounts Dashboard
- Transactions Dashboard
- Loans Dashboard
- Cards Dashboard
- Complaints Dashboard

- ## SQL Analysis

Advanced SQL queries were used to calculate key banking KPIs including:

- Active Customers
- Branch-wise Deposits
- NPA Ratio
- Complaint Resolution Rate
- Monthly Transaction Trends

## Project Outcome

This project demonstrates end-to-end data analytics including database design, data validation, advanced SQL analysis, and interactive dashboard development.
